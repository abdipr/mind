# Mind

This isn't just documentation.

It's where I store the things I learn — from small insights to the ones that fundamentally shift how I think.

If you’ve stumbled upon this page, think of it as a peek inside my head, organized enough for others to read.

## Why did I build this?

Because I forget. A lot.

And sometimes, things I’ve learned the hard way… they just vanish if I don't write them down.

## How to use this

You don't have to read this in order.

But if you're not sure where to start:
- Start with **Life**
- Move on to **Thinking**
- Then dive into **Tech**

## A quick note

Things here will likely change over time.

And that’s exactly the point.
