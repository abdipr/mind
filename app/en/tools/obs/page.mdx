# OBS Studio

To keep recordings smooth and audio clean, these are the standard settings I use.

## Audio

- Use the built-in **noise suppression** (RNNoise) for the mic.
- Separate desktop audio and mic into different tracks for easier editing.

## Video

- Use bitrate above **6000 Kbps** for 1080p screen recording so text stays sharp.
- Encoder: NVENC if you have Nvidia GPU, x264 if CPU is strong.

## Tips

- Always test record 30 seconds before starting a long session.
- Set hotkeys for start/stop recording so you don't need to switch windows.
