# FL Studio

Tempat kabur dari kode. Bikin beat lofi atau sekadar eksperimen sama synth.

## Tips Mixing

- Selalu **sidechain** kick ke 808 biar low end-nya nggak numpuk.
- Jangan terlalu banyak taruh plugin di Master channel sebelum mixing selesai.
- EQ cut dulu, baru boost. Subtractive sebelum additive.

## Plugin Favorit

- **Serum** — Synth serba bisa
- **Valhalla Supermassive** — Reverb gratis yang gokil
- **iZotope Vinyl** — Efek lo-fi instan, gratis
