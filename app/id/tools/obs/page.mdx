# OBS Studio

Biar rekaman mulus dan audio nggak bocor, ini setting standar yang biasa gue pake.

## Audio

- Gunakan **noise suppression** bawaan OBS (RNNoise) buat mic.
- Pisahkan audio desktop dan mic ke track berbeda biar gampang editing.

## Video

- Pake bitrate di atas **6000 Kbps** buat rekaman layar 1080p biar teksnya tetep tajam.
- Encoder: NVENC kalo GPU Nvidia, x264 kalo CPU kuat.

## Tips

- Selalu test rekaman 30 detik sebelum mulai session panjang.
- Set hotkey buat start/stop recording biar nggak perlu switch window.
